## Critical Anti-Patterns (advisory — each discovered by actual failure this session)

| Pattern | Description | Severity | Prevention |
|---------|-------------|----------|------------|
| Body-scroll in this SPA | App content scrolls inside `<main class="overflow-y-auto">`, NOT `<body>`. Playwright `window.scrollTo` no-ops. | advisory | Scroll `document.querySelector('main')` (`m.scrollTop = m.scrollHeight`) to screenshot below the fold. |
| Brand-guide lockup naming | `icon-dark-text.png` has a **WHITE** wordmark (for dark bg); `icon-text.png` is the **dark/navy** wordmark (for light bg). Names are misleading. | advisory | Light-bg lockup = `ph-lockup-light.png` (from `icon-text`); dark-bg = `ph-lockup-dark.png` (from `icon-white-text`). |
| Undefined Tailwind token | `text-neutral-800` isn't defined in this theme (only 0/50/100/…/700/900) → never flips in dark. | advisory | Use only defined neutral tokens; already fixed repo-wide. |
| Fabricating DOIs | Mimno 2011 (`D11-1024`) & Newman 2010 (`N10-1012`) predate ACL DOI assignment; `10.18653/v1/…` variants 404. | advisory | Link canonical `aclanthology.org/…` pages; do NOT invent DOIs. |
