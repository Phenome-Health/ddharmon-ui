## Critical Anti-Patterns (each discovered by an actual failure/finding this session)

| Pattern | Description | Severity | Prevention |
|---------|-------------|----------|------------|
| API key in the config body | The BYOK key must NOT go in the `config` JSON — that object is persisted as the job's `run_config` (`store.create`), so it would leak to disk/logs. | blocking (for the backend) | Key rides as a transport-only `x-anthropic-key` header only; held in memory for the job, never persisted. |
| Mutating `os.environ` for per-request keys | Core `anthropic.Anthropic()` reads `ANTHROPIC_API_KEY` from env in **4 places** (`src/ddharmon/llm/batch.py` ×3, `llm/anthropic_client.py:64`). Setting env per-request races + leaks keys across concurrent jobs. | blocking (for the backend) | Thread the key down to each client constructor as `api_key=...`; never touch process env. |
| Tailwind arbitrary `clamp()` minus operator | `clamp(1rem,16vh-117px,3.5rem)` is invalid CSS — `calc`-style math needs spaces around `-`. | advisory (fixed) | Use underscores in the arbitrary value: `lg:py-[clamp(1rem,16vh_-_117px,3.5rem)]` → Tailwind expands `_`→space. |
| Force-fit-to-viewport layout | Pinning a multi-section page to 100vh / `flex-1` fill is fragile across zoom + browser chrome and needs magic pixel offsets. | advisory | Use fluid `clamp()` spacing keyed to vh; floors = the compact case, capped maxima. Let it scroll on genuinely tiny windows. |
| Playwright module resolution | `import "playwright"` from a scratchpad script fails — ESM resolves from the script's dir, and Playwright is only in `frontend/node_modules`. | advisory | Put screenshot scripts inside `frontend/` (or run from there) so the bare import resolves. |
