| Pattern | Description | Severity | Prevention |
|---------|-------------|----------|------------|
| GitHub Pages "Deployment failed, try again later" | `actions/deploy-pages@v5` deploy step fails GitHub-side (build finds the artifact fine). | Static build must set wouter base | SPA on a subpath needs `<Router base>` or routes don't match. | advisory | Already done: `App.tsx` uses `import.meta.env.BASE_URL`. |
