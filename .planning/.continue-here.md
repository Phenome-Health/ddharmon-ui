## Critical Anti-Patterns (each discovered by an actual failure this session)

| Pattern | Description | Severity | Prevention |
|---------|-------------|----------|------------|
| Space-joined composite map keys | `buildSankeyData` keyed maps `` `${v} ${dest}` `` and split on `" "`, but destination names contain spaces (`"Existing CDE"`) ‚Üí mis-parsed node name ‚Üí **undefined Sankey link target ‚Üí recharts layout crash** (`centerY` reads `undefined.y`). | advisory (fixed) | Keys are now JSON tuples (`JSON.stringify([v, dest])`). Never key composite maps by a separator that can appear in the values. |
| recharts mutates your data | recharts Sankey resolves `link.source/target` indices into node objects and writes `x/y/depth` onto nodes **in place**. | advisory | Hand recharts a throwaway clone each render (`{nodes: nodes.map(n=>({...n})), links: links.map(l=>({...l}))}`); keep the canonical copy for your own index math. |
| Stale static fixtures after a seed change | The atlas points were synthetic (`{cohort}:v{i}`) unlinked to records, so verdict-color + click-through showed nothing. | advisory | `backend/seed.py::_atlas` now emits one point per real member. To regen just the `result-*.json` **without a running backend**, import `backend.seed._atlas` and rewrite `d["result"]["atlas"]` in each committed fixture (the HTTP `build_static_fixtures.py` needs the server up + also rebuilds exports). |
| SPA scrolls inside `<main>` | `window.scrollTo` / Playwright `fullPage` misbehave (body doesn't scroll). | advisory | Scroll `document.querySelector('main')`, or flatten it (`m.style.height='auto';overflow='visible'`) before a full-page screenshot. |
| Verifying deployed bundles | BSD `grep` treats the minified JS as binary and "fails" even on plain words; `curl` without `--compressed` scans compressed bytes. | advisory | Use `curl --compressed` + `grep -a`, or (best) read the rendered DOM via Playwright. |
| Raw `>` in JSX prose | A literal `-->` in hero copy broke the esbuild build (`TS1382`). | advisory | Use `‚Üí`/entities; the deploy script now runs `tsc` before building so this aborts locally, never ships. |
